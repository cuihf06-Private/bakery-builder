# BREAD-BAKERY-GAME

一个基于 HTML/JavaScript 的面包烘焙模拟小游戏。

## 项目简介
本项目为单页面小游戏，玩家通过经营面包店，不断升级、挑战更高分数。

## 主要特性
- 回合制烘焙与经营
- 升级系统与商店
- 简洁的界面与交互

## 快速开始
1. 克隆仓库到本地：
   ```
   git clone https://github.com/你的用户名/BREAD-BAKERY-GAME.git
   ```
2. 直接用浏览器打开 `index.html` 即可游玩。

## 目录结构
- `index.html`：主页面及全部游戏逻辑
